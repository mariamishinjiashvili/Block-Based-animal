# WebGL Animated Dog Model

## Overview
This project creates a 3D animated dog model using **WebGL**. The dog is built from multiple cube primitives and demonstrates concepts including 3D transformations, hierarchical modeling, lighting, shaders, and interactive animation.

The dog is designed to stand on its rear legs and includes animated legs and body movement controlled through keyboard input.

## Features
- Interactive controls for:
  - Moving legs
  - Adjusting the dog's standing angle

## Technologies Used

- WebGL
- JavaScript
- GLSL Shaders
- HTML5 Canvas
- Matrix transformations
